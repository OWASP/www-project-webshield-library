# OWL Core Node Demo

Runnable Node example showing how to compose the framework-agnostic `@owl/core` APIs.

## Run

From this folder:

```bash
npm start
```

## What it demonstrates

- A01 role and ACL policy decisions
- A03 request validation and sanitization
- A02 encryption before transport
- A07 session and token lifecycle
- A08 secure HTTP request composition
- A05/A06 operational checks and reporting
- A09 structured logging

The script uses deterministic mocks so it can run locally without external services.
