# Support

## Getting Help

Use these paths for help:

- Documentation and examples: README and docs folder
- Non-security bug reports: open a public issue
- Security vulnerabilities: follow SECURITY.md

## Before Opening an Issue

- Confirm behavior on latest code.
- Run:

```bash
npm run check
```

- Include minimal reproduction and expected behavior.

## Maintainer Expectations

Best-effort support is provided. Response time depends on maintainer availability.
